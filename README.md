#  Індивідуальне завдання: Алгоритми на графах

##  Опис проєкту
У проєкті реалізовано модель частини території України у вигляді графа.  
Граф включає 20 міст (Вінниця, Київ, Умань та ін.) та понад 30 доріг між ними.

##  Реалізовані алгоритми

- **DFS (пошук у глибину)** — обхід усіх міст  
- **BFS (пошук у ширину)** — знаходження шляху без урахування ваг  
- **Алгоритм Дейкстри**:
  - найкоротший шлях за відстанню (км)
  - оптимальний шлях за типом дороги  
- **Сортування**:
  - за довготою (із заходу на схід)
  - за населенням (за спаданням)

##  Характеристики графа

- **20 вершин** (міст)
- **31 ребро** (доріг)

###  Ваги ребер:
- `a` — відстань у кілометрах  
- `b` — тип дороги:
  - `1` — міжнародна (найкраща)
  - `2` — регіональна
  - `3` — місцева (нижчого типу)
